# 🤖 ZXTR-AI
**Sistem AI Lokal Mandiri 100% Offline**

ZXTR-AI adalah asisten kecerdasan buatan yang berjalan sepenuhnya di perangkat kamu sendiri, tanpa memerlukan koneksi internet atau mengirim data ke pihak ketiga. Mendukung berbagai model GGUF populer, bisa digunakan untuk membuat kode, menjawab pertanyaan, dan banyak lagi.

---

## ✨ Fitur Utama
✅ **100% Lokal & Aman**: Semua proses berjalan di perangkat, data tidak keluar
✅ **Dukungan Banyak Model**: Otomatis mendeteksi & bisa beralih antar model:
- Mistral-7B (ringan, cepat untuk HP)
- Llama 3.1 70B (kualitas tinggi)
- Qwen2-72B (terbaik untuk kode & bahasa Indonesia)
- Mixtral 8x7B (seimbang kecepatan & kualitas)
✅ **Bisa Apa Saja**: Buat kode, jawab pertanyaan, jelaskan konsep, bantu perencanaan
✅ **Antarmuka Web**: Mudah diakses lewat browser
✅ **API Kompatibel**: Bisa dipakai seperti API OpenAI untuk proyek lain
✅ **Gratis Selamanya**: Tidak ada biaya langganan
